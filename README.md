Note, this repo uses (pnpm)[https://pnpm.io/]

## Getting Started

Install

```bash
pnpm install
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Testing

```bash
pnpm run test
```
